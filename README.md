## Project Overview

This project analyzes employee attrition patterns using the IBM HR Analytics dataset, demonstrating:
- Advanced Python analysis (Pandas, NumPy, Matplotlib, Seaborn)
- SQL database analytics (Complex queries, CTEs, Window Functions)
- Tableau dashboards (Interactive visualizations)
- Power BI reports (DAX measures, Power Query)

### Key Findings
- Overall attrition rate: 16.12%
- 3x higher risk for employees with low satisfaction
- $3.56M annual cost of attrition identified
- 87 critical-risk employees requiring intervention
- $1.13M potential savings with targeted strategies

## Analysis Highlights

### 1. Satisfaction Analysis
- Low satisfaction: 28.9% attrition
- High satisfaction: 9.2% attrition
- 3x higher risk for dissatisfied employees

### 2. Work-Life Balance
- Overtime workers: 30.5% attrition
- Non-overtime: 10.4% attrition

### 3. Career Progression
- Stagnant promotions (>5yr): 25.6% attrition
- Recent promotions: 10.2% attrition

### 4. Compensation Equity
- Bottom 25% income: 23.4% attrition
- Top 25% income: 11.8% attrition

### 5. Risk Profiling
- 87 critical-risk employees identified
- 45.7% attrition in critical category
